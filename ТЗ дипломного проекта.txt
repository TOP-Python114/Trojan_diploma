Приложение по размещению вакансий организации.

Цель: Предоставление соискателям информации о имеющихся в организации открытых вакансиях, предоставление соискателям возможности направить резюме на определённую должность.

В приложении предусмотрена аутентификация пользователя.

- Роли (группы) пользователей:
    1. Суперпользователь
    2. Администратор с правами доступа (редактор) - добавлять, редактировать, удалять вакансии.
    3. Пользователь (просмаотривает открытые вакансии, имеет возможность откликнуться на определенную вакансию)
    4. Анонимный пользователь (просматривает открытые вакансии без возможности откликнуться на вакансию)

- "Главная страница" содержит (для пользователя и анонимного пользователя):

    - Шапка сайта:
        Название организации на фоне фото техники организации
        Ссылка на форму аутентификации и регистрации

    - Тело страницы:
        Список открытых вакансий (ссылка на страницу с полным описанием вакансий)
        Логотип
        Краткое описание организации
        Карта с местоположением организации
        Контактные данные

- "Главная страница" содержит (для администратора/редактора):

    - Шапка сайта:
        Название организации на фоне фото техники организации
        Ссылка на форму аутентификации и регистрации

    - Тело страницы:
        Кнопка "добавить вакансию" (переход на страницу с формой для добавления вакансии)
        Список открытых вакансий (ссылка на страницу с полным описанием вакансий)
        Логотип
        Краткое описание организации
        Карта с местоположением организации
        Контактные данные

- Страница  "Вакансии" содержит (для анонимного пользователя):

    - Шапка сайта:
        Название организации на фоне фото техники организации
        Аутентификация пользователя

    - Тело страницы:
        Полное описание вакансий
        Карта с местоположением организации
        Контактные данные

- Страница  "Вакансии" содержит (для пользователя):

    - Шапка сайта:
        Название организации на фоне фото техники организации
        Аутентификация пользователя

    - Тело страницы:
        Полное описание вакансий
        Кнопка "откликнуться" (для каждой вакансии. После чего открывается почта пользователя,
        в тексте письма автоматически формируется сообщение с отсылкой на определенную вакансию, письмо автоматически отправляется на почту организации)
        Карта с местоположением организации
        Контактные данные

- Страница  "Вакансии" содержит (для администратора/редактора):

    - Шапка сайта:
        Название организации на фоне фото техники организации
        Аутентификация пользователя

    - Тело страницы:
        Блоки с полным описанием вакансий - кнопка "редактировать", кнопка "удалить"
        Карта с местоположением организации
        Контактные данные
